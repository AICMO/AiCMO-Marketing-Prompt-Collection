## Integration Tips
- **Google Ads:** Export campaign performance via Reports > Predefined Reports > Time > Last 30 days. Download as CSV and paste the top-level summary into the Quick Version. For impression share: use the Auction Insights report.
- **Meta Ads Manager:** Use Columns > Customize Columns > add Frequency, Reach, Unique CTR, and Cost per Result. Export as CSV. Check creative age in the Ads tab sorted by Date Created.
- **LinkedIn Campaign Manager:** Export via Reports > Performance Report. Add Click-through Rate, Impressions, Conversions, and Cost per Conversion columns.
- **Google Looker Studio:** Connect all three platforms via native connectors and build a cross-channel blended ROAS dashboard that auto-refreshes daily. Paste the dashboard summary into the Advanced Version for faster weekly analysis cycles.
- **HubSpot:** Use the Ads Attribution Report (Marketing > Ads > Attribution) to pull closed-revenue ROAS instead of platform-reported revenue. Platform numbers typically run 3-5x higher than CRM-verified revenue due to attribution window inflation.
- **Zapier / Make:** Automate a weekly data pull from Google Ads, Meta, and LinkedIn into a Google Sheet. Trigger a scheduled prompt run every Monday morning to produce a weekly performance diagnostic without manual data assembly.

## Troubleshooting

**Problem: Platform ROAS numbers do not match what I see in my CRM or HubSpot.**
Solution: This is almost always an attribution window mismatch. Google Ads defaults to a 30-day click window; Meta defaults to 7-day click + 1-day view. Set all platforms to the same window (7-day click, no view-through) before running this analysis to get comparable cross-channel numbers. For B2B with long sales cycles, run an assisted revenue report from your CRM and treat that as ground-truth revenue.

**Problem: I do not have conversion tracking set up on LinkedIn -- CPA shows zero.**
Solution: Install the LinkedIn Insight Tag via Google Tag Manager and set up a conversion event on your thank-you page or demo confirmation page. Until tracking is live, use click-to-lead rate (Leads / Clicks) from LinkedIn Lead Gen Forms as a proxy metric. Do not allocate more than minimum viable spend ($5K/month) to any channel without verified conversion tracking.

**Problem: The reallocation recommendations conflict with channel minimums set by leadership.**
Solution: Add explicit budget floor constraints to the Advanced Version prompt ("Minimum spend per channel: Google $X, Meta $X, LinkedIn $X"). The model will optimize within those constraints and quantify the efficiency cost of each floor in dollar terms -- use that number to make the business case for removing artificial minimums at your next budget review.
