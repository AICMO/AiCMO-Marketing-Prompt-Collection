## Integration Tips

**HubSpot:**
- Create a custom Social Media Performance dashboard using the Social Reports module
- Use UTM parameters `utm_source=[platform]&utm_medium=organic-social&utm_campaign=[content-series]` on all bio and post links to track contact origination in HubSpot contacts
- Set up a HubSpot workflow: when a contact's original source = "Social Media," enroll them in a tailored nurture sequence

**Google Analytics 4:**
- Build a custom report: Session source/medium filtered to organic social channels
- Use GA4 Explorations → Path Exploration to trace the exact social → conversion journey
- Set up a GA4 audience segment for "Social Engagers" (visited from social 2+ times in 30 days) and push to Google Ads for retargeting

**Sprout Social / Metricool / Buffer Analytics:**
- Export the "Post Performance" CSV from any of these tools and paste directly into the Advanced Version prompt
- Sprout's "Competitor Reports" tab provides the engagement benchmarks needed for Section 4
- Metricool's "Best Time to Post" heatmap validates or challenges the posting time recommendations

**Notion:**
- Build a Content Performance Database with properties: Platform, Date, Format, Topic Category, Impressions, Engagement Rate, Saves, Link Clicks, Health Score
- Run this prompt monthly and paste Section 1 Executive Dashboard into a Notion database for trend tracking

**Zapier / Make (Integromat):**
- Automate weekly social metric pulls: Social platform → Google Sheets → trigger this prompt with the new data → output delivered to Slack #marketing channel
- Workflow: New buffer analytics report available → Zapier extracts CSV → passes to Claude API with this prompt → posts summary to Slack

**Looker Studio:**
- Connect native platform APIs or a tool like Supermetrics to feed live social data
- Build a single dashboard with: follower growth trend, engagement rate by platform, top posts by engagement, content format performance split, and social-attributed website sessions
- Set automated email delivery to CMO every Monday at 8am

## Troubleshooting

**Problem: Engagement rate looks healthy but no pipeline is being generated from social.**
Fix: The issue is likely a conversion path gap, not a content quality problem. Check whether bio links use UTM-tracked URLs that route to a high-converting landing page (not the homepage). Run this prompt's Attribution section and audit whether any posts contain a clear call-to-action with a specific next step. Add one CTA post per week that directly asks the audience to take an action (book a demo, join the waitlist, download the guide).

**Problem: Follower growth is good but engagement rate is declining.**
Fix: This is classic audience quality dilution — rapid follower growth often brings in non-ICP followers who don't engage. Run the Audience Intelligence section and check whether recent growth was driven by viral off-topic content. If so, temporarily reduce broad-appeal content and increase ICP-specific content (niche topics, industry jargon, specific pain points) to re-calibrate the algorithm feed toward your actual buyer.

**Problem: Data is incomplete — only have follower counts and likes, no impression or click data.**
Fix: Even with partial data, run the Quick Copy-Paste Version with what you have and note the gaps explicitly. Immediately set up native analytics access for all platforms (LinkedIn Analytics, Instagram Professional Dashboard, TikTok Creator/Business Center, X Analytics) — all are free. Install UTM tracking on all bio links today using a link-in-bio tool like Linktree with UTM parameters built in. Prioritize getting complete data for the next reporting cycle before optimizing further.
