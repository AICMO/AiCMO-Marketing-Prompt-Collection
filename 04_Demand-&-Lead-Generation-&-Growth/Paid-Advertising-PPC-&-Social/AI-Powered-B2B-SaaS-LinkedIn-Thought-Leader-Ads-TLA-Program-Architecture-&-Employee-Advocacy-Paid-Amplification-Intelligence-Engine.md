## Integration Tips

- **LinkedIn Campaign Manager API:** Use the LinkedIn Marketing API to automate TLA post sponsorship — POST to `/adCreativesV2` endpoint with `type: THOUGHT_LEADER_ADS`; trigger automation when a post crosses your engagement threshold using a Zapier/Make webhook on LinkedIn activity
- **HubSpot:** Map LinkedIn TLA as a UTM source in contact attribution; create a "TLA Influenced Pipeline" report by filtering opportunities where original source or any-touch includes `utm_medium=tla-paid`; set up a workflow that tags contacts touched by TLA with "LinkedIn TLA Influenced" property for segmentation
- **Salesforce:** Create a custom Campaign object for each TLA campaign; use Salesforce's Campaign Influence (Multi-Touch Attribution) to track TLA touches across the pipeline; build a dashboard showing "LinkedIn TLA Pipeline Influence by Quarter"
- **Notion/Confluence Content Calendar:** Build a TLA content queue in Notion with Author, Post URL, Post Score, Campaign Assignment, Boost Date, and Performance columns; connect to LinkedIn API via Zapier to auto-pull organic post metrics daily
- **Slack Alerts:** Set up a Zapier/Make automation that posts to #marketing-wins Slack channel when a TLA post exceeds 50K impressions or generates a demo request — creates internal visibility and motivates authors to keep posting
- **Google Sheets Pipeline Tracking:** Export LinkedIn Campaign Manager data weekly via API to Google Sheets; use VLOOKUP against CRM opportunities to calculate TLA-influenced ARR; share with CFO/CRO for monthly pipeline review

## Troubleshooting

**Problem: TLA posts are approved by LinkedIn but not spending — campaigns stuck in "Learning" or "In Review"**
Solution: LinkedIn Thought Leader Ads require the post author to approve boosting — ensure you have proper authorization set up. The author must be listed as an admin on the associated Page, and they must accept the TLA permission request via LinkedIn notification. If stuck in "Learning," give campaigns 48-72 hours minimum; if still not delivering after 5 days, check that your audience size is >50,000 members (LinkedIn minimum for delivery) and try broadening targeting.

**Problem: Organic posts don't qualify for TLA (scoring below threshold) but we need content to boost this week**
Solution: Run a 48-hour "organic seeding" strategy — have the author post on Tuesday, then signal-boost by having 10-15 employees engage (like, comment, share) within the first 2 hours. LinkedIn's algorithm rewards early engagement velocity; a post that gets 40+ reactions in 2 hours often reaches 3-5x the organic baseline. Then re-score on Wednesday and boost if threshold is met. Alternative: Don't wait for organic — write a TLA-optimized post specifically designed to be boosted, seed it organically for 24 hours, then activate TLA.

**Problem: CPL from TLA is higher than branded Sponsored Content — hard to justify TLA budget to leadership**
Solution: TLA's real value is in pipeline influence and CAC reduction over time — not last-touch CPL. Pull a 90-day cohort analysis showing: (a) deals that had TLA impressions vs. deals with zero TLA touchpoints — compare average sales cycle length and deal size; (b) calculate "assisted pipeline" — deals where TLA was a touchpoint but not the last before conversion. Present "TLA Pipeline Influence" as the primary metric ($X of pipeline touched by TLA per $1 of spend) rather than CPL. If CPL is still the requirement, shift budget to Campaign 4 (direct response to high-intent retargeting) where CPL should be $200-400 for most B2B SaaS companies.
