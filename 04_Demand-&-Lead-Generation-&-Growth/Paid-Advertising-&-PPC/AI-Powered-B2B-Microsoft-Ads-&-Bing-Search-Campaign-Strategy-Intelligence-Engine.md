## Troubleshooting

**Problem: LinkedIn Profile Targeting audiences are too small (under 1,000 users)**
- Fix: Expand company size range (e.g., 100–10,000 instead of 500–5,000), add 1–2 additional job functions, or remove the seniority filter and use bid adjustments instead. Microsoft requires minimum audience sizes for targeting; if under threshold, the filter won't apply.

**Problem: Microsoft Ads Quality Scores are low (1–4) driving up CPCs**
- Fix: The three QS factors are expected CTR, ad relevance, and landing page experience. Run Ad Relevance diagnostic first — ensure the exact keyword appears in the RSA headline 1 or 2. For landing page experience, use Microsoft's ad preview tool to check load speed and mobile responsiveness. Consider creating tighter ad groups (5–10 keywords max) with dedicated landing pages per theme.

**Problem: Conversion tracking shows 0 conversions despite form submissions confirming leads**
- Fix: Use Microsoft's UET Tag Helper Chrome extension to verify the tag fires on thank-you pages. Check if your form uses a redirect vs. same-page AJAX confirmation — AJAX forms require custom event tracking (not page-load triggers). Verify the conversion goal in Microsoft Ads matches the UET event name exactly (case-sensitive). Test in Microsoft Clarity (free heatmap tool) to confirm the tag fires on real sessions.
