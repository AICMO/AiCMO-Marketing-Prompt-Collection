## Troubleshooting

**Problem: Low delivery — ads not spending budget**
*Cause:* Audience too small (<50K) or bid too low for auction competition.
*Fix:* Expand audience by adding more follower lookalike accounts (aim for 500K+ per ad group), increase max bid by 20-30%, or switch from Manual to Auto Bid. Check if keyword targeting is too narrow — broaden with adjacent terms.

**Problem: High CTR but zero conversions (leads or purchases)**
*Cause:* Message mismatch between ad and landing page, or wrong audience intent stage.
*Fix:* Verify landing page H1 mirrors the ad headline exactly. Check if the traffic is mobile (if so, is the form mobile-optimized?). Switch to X Lead Gen Cards to remove landing page entirely. Audit audience — keyword targeting should be higher intent than follower lookalike; if follower lookalike is converting worse, pause and reallocate.

**Problem: X Pixel showing zero conversions despite traffic**
*Cause:* Pixel misfired, wrong event tracking, or iOS/browser blocking attribution.
*Fix:* Use X Pixel Helper Chrome extension to verify pixel fires on each relevant page. Implement Conversion API (CAPI/server-side) to capture conversions blocked by browser privacy settings. Verify the conversion event in X Ads Manager matches the event name fired on the page exactly (case-sensitive). Set attribution window to 3-day click if 1-day is returning zero (some consideration-stage buyers convert on Day 2-3).
