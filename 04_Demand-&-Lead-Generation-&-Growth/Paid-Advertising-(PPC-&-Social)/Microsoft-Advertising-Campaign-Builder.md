## Integration Tips

- **Microsoft Advertising Editor:** Export the campaign structure from this prompt directly into Editor's bulk import CSV format. Use the keyword table output to populate the "Keyword," "Match Type," and "Max CPC" columns.
- **Google Ads Import:** Use Microsoft's native Google Ads import tool, then apply the 10-point migration checklist from the Advanced Version to clean up mismapped settings, network defaults, and consumer-intent negatives.
- **HubSpot / Salesforce CRM:** Connect offline conversions via Microsoft's Offline Conversion API. Map CRM lifecycle stages (MQL → SQL → Closed Won) to Microsoft's conversion goal framework for revenue-attributed reporting.
- **Zapier / Make:** Automate weekly performance pulls from the Microsoft Ads API into Google Sheets dashboards. Use the KPI list from the Measurement section as column headers.
- **LinkedIn Campaign Manager:** Cross-reference LinkedIn Profile Targeting audience definitions with active LinkedIn Ads audiences to ensure consistent messaging across both platforms for the same segments.
- **UET Tag Manager:** Deploy the Universal Event Tracking tag via Google Tag Manager using the "Microsoft Advertising" trigger template—no code needed for most CMS platforms.

## Troubleshooting

**Problem: Google Ads import created keyword conflicts and inflated CPCs**
Solution: After import, run the "Search Terms" report for the first 7 days and add all irrelevant consumer-intent terms as exact match negatives. Common culprits: "free," "download," "tutorial," "how to." Also disable the Microsoft Search Partner Network for the first 30 days until baseline Quality Scores stabilize.

**Problem: LinkedIn Profile Targeting audiences are too small (under 1,000 users) and campaigns won't serve**
Solution: Broaden company size ranges first (e.g., expand 100–500 to 50–1,000), then loosen job title restrictions to include adjacent titles. If still under threshold, switch from "AND" targeting logic to "OR" across job function + company size. Minimum viable audience is 1,000 monthly actives on the Audience Network.

**Problem: Conversion tracking shows zero conversions despite traffic**
Solution: Verify UET tag fires on the thank-you/confirmation page (not the form page) using Microsoft Clarity's event viewer or the UET Tag Helper Chrome extension. Ensure the conversion goal is set to "Destination URL" or "Custom Event" matching your thank-you page URL exactly, including any trailing slashes or query parameters.
