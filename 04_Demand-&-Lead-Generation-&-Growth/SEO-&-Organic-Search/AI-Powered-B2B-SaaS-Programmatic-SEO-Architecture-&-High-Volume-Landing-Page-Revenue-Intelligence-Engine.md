## Overview

## Integration Tips

**Webflow CMS / HubSpot CMS:**
Export your programmatic page data schema to a CMS Collection. Use CMS templating to auto-generate pages from collection items. Vaultly's Salesforce integration pages, for example, can live as individual CMS items in a "Integrations" collection with fields matching the data schema above.

**Airtable + Make (Zapier alternative):**
Build your data schema as an Airtable base. Use Make automations to: (1) trigger on new row creation, (2) populate the CMS template via API, (3) publish the page, (4) submit URL to Google Search Console Indexing API. Full zero-touch page creation pipeline.

**Google Search Console API:**
After publishing each batch of programmatic pages, call the GSC Indexing API to request crawling. Reduces time-to-index from weeks to hours for new pages. Python script for bulk submission available via open source tools.

**Screaming Frog + Ahrefs:**
Schedule monthly Screaming Frog crawls of your programmatic page directory. Export to Ahrefs Batch Analysis to check: indexed status, ranking keywords per page, organic traffic per page. Flag pages with zero impressions after 60 days for content quality review or consolidation.

**HubSpot CRM Attribution:**
Create a HubSpot custom property "First Touch Programmatic Page" using the UTM parameter or page URL stored on contact record. Build a report showing programmatic SEO page category → pipeline value attribution. Justifies continued investment to CFO.

**Salesforce / Pardot:**
If programmatic page converts via form, pass the page URL as a hidden field into the lead record. In Salesforce, create a report grouping leads by programmatic page category to see which archetypes (integration vs. use case vs. competitor) generate highest-ACV opportunities.

## Troubleshooting

**Problem: Google is not indexing programmatic pages despite sitemap submission.**
Fix: Check that pages have sufficient unique content (minimum 400 words of non-template-filler content). Use "noindex" on any pages where the variable data fields are empty or produce fewer than 200 words of unique text. Request indexing via Google Search Console URL Inspection for your highest-priority 25 pages manually. Ensure no accidental `robots.txt` disallow for the programmatic URL directory.

**Problem: Programmatic pages are ranking but not converting — lots of impressions, zero demo requests.**
Fix: Audit CTAs for intent-alignment mismatch. Integration pages attract current users looking for setup help — add a secondary CTA like "Upgrade to unlock [integration]" or "Talk to our integration team." Comparison pages should have aggressive CTAs since that audience is actively buying; use "Switch from [Competitor] in 14 days — see how" rather than generic "Book a demo."

**Problem: Google is treating programmatic pages as duplicate content and consolidating them in search.**
Fix: Increase content differentiation between pages. Each template must include at least 150-200 words of genuinely unique content for that specific variable (e.g., the Salesforce integration page discusses Salesforce-specific objects, not generic "your CRM"). Add a customer quote unique to that page. Vary FAQ questions between closely-related pages (integration A vs. integration B). If two pages are truly too similar (e.g., two integrations that do identical things), consolidate into one page covering both.
