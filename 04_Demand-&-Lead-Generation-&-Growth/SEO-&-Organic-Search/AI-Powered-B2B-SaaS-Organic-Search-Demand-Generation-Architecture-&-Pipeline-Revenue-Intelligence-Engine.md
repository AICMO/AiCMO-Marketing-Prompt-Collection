## Integration Tips

- **HubSpot:** Create a custom "Original Source Drill-Down 2" property mapped to utm_campaign to track which content cluster each contact came from; build a report filtering for contacts where "Original Source = Organic Search" to see organic MQL trends by cluster
- **Salesforce:** Use the Campaign object to represent each topic cluster; create Campaign Members when organic visitors convert; run an Opportunity Influence report to see pipeline by content cluster
- **Google Search Console API:** Automate weekly keyword ranking reports using the GSC API + Google Sheets or a BI tool; set up alerts for position drops >5 on any keyword in your top 50
- **Ahrefs / Semrush API:** Use Ahrefs Rank Tracker API or Semrush Position Tracking API to monitor keyword ranking changes daily; pipe results into a Slack alert for position entries in top 10
- **Webflow CMS:** Use Webflow's CMS API to programmatically publish AI-generated article drafts after human review; tag each article with content cluster and buyer journey stage for automated reporting
- **Zapier:** Build a Zap that triggers when a new organic contact is created in HubSpot → assigns to the correct content cluster → updates a Google Sheets tracking dashboard → notifies the content team via Slack

## Troubleshooting

**Problem:** Published articles are not ranking after 60+ days despite targeting low-difficulty keywords.
**Solution:** Check three root causes: (1) Crawlability — verify the page is indexed in Google Search Console (URL Inspection → "URL is on Google"); if not indexed, fix crawl blocks or canonicalization issues. (2) Topical authority — Google may not yet trust your domain on this topic; publish 5+ additional cluster articles on the same topic before expecting the pillar to rank. (3) Content quality — run the top-ranking pages through a content gap analysis tool and identify missing subtopics, semantic keywords, or schema markup your article lacks.

**Problem:** Organic traffic is growing but MQL conversion rate from organic is below 1%.
**Solution:** The content is attracting wrong-intent visitors. Audit your top 10 organic landing pages: (a) Are the CTAs aligned to the keyword intent? A "how to" guide should offer a template or checklist, not a demo request. (b) Is your ICP actually searching these terms? Run a search for your top keywords and check if the SERP results suggest the searcher is a decision-maker or a practitioner. (c) Add intent-specific CTAs: high-funnel articles → newsletter or template download; mid-funnel → case study or ROI calculator; low-funnel → demo or free trial. Track CTA conversion rate by article to identify which content actually converts your ICP.

**Problem:** AI-generated content drafts are flagged as low-quality during expert review — too generic, wrong voice, missing ICP-specific insights.
**Solution:** Improve the AI brief inputs: (1) Include 3–5 verbatim quotes from customer interviews or sales call transcripts that describe the problem in the buyer's own language — instruct the AI to use this language throughout. (2) Add a "Voice and POV" section to every brief that specifies your company's differentiated perspective on the topic (not just restating what the article is about). (3) Feed the AI your top 3 highest-performing existing articles as style references. (4) Require the AI to include at least one specific, falsifiable claim (a statistic, a benchmark, or a named customer outcome) in every article section — generic content contains no verifiable facts.
