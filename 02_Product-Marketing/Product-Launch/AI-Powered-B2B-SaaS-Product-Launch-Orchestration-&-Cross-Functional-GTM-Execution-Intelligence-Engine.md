Use it when preparing any product launch where misalignment between Product, Marketing, Sales, CS, and Support would cost revenue.

## Integration Tips

- **HubSpot / Marketo / Pardot:** Use the launch email templates directly as workflow email bodies. Set up a launch-specific campaign tag to track pipeline influenced across all touchpoints. Create smart lists segmented by ICP fit score to prioritize customer outreach order.
- **Salesforce:** Build a launch campaign object with start date = launch day. Tag all opportunities influenced by launch outreach. Use the 30/60/90-day metrics as Salesforce report filters.

## Troubleshooting

**Problem: Cross-functional teams keep missing deadlines and the launch is slipping.**
Solution: Implement a mandatory weekly launch sync (30 minutes, all workstream leads) starting at T-8. Use the readiness tracker as the agenda — each owner gives a 1-minute GO/NO-GO status. Any NO-GO triggers an immediate problem-solving conversation, not an async Slack thread. If a workstream is chronically red, escalate to their VP with a specific ask: "We need X by [date] to launch on [date]. What resources do you need?"

**Problem: Sales team isn't using the launch assets and is going off-script in deals.**
Solution: Record a 10-minute video walkthrough of the messaging brief and objection handler. Embed it in the CRM as a Pinned Note or Chatter post. Set up a Salesforce/HubSpot task that fires when a rep opens an opportunity tagged with the new product — the task links to the asset library. Run a 30-minute live role-play session where reps practice the objection handler before launch. Follow up 2 weeks post-launch with call recording review (Gong / Chorus) to identify which messages are resonating and which are being skipped.

**Problem: Post-launch metrics look flat — traffic and demo requests aren't moving.**
Solution: Diagnose the funnel in order: (1) Traffic — did paid campaigns go live? Is organic content indexed? (2) Landing page CVR — is the page loading fast? Is the CTA above the fold? Run a 24-hour heatmap with Hotjar. (3) Email deliverability — check open rates vs. send volume; if <15% open rate, check spam score and subject line variants. (4) Message-market fit — pull the last 5 sales call recordings and look for the moment the prospect disengages. If the headline isn't landing in the first 30 seconds of a demo, rewrite it using customer language from win/loss interviews.
