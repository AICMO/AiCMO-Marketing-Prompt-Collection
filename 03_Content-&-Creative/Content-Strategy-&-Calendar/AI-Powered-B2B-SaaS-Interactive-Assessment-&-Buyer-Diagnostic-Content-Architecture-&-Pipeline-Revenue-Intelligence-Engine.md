## Integration Tips

**Typeform + HubSpot (most common stack):**
- Use Typeform's "score" field to calculate dimension sub-scores natively
- Connect Typeform → HubSpot via native integration; map each answer to a custom HubSpot property
- Create HubSpot Workflows triggered by [Assessment Tier = X] that enroll contacts into the appropriate email sequence
- Use HubSpot's custom object to store full assessment response data alongside the contact record for sales visibility

**Salesforce + Marketo:**
- Use Typeform or Survey Monkey Enterprise; push results to Marketo via Zapier or direct API
- Create Marketo Smart Campaigns triggered by custom field "Assessment Tier" (values: At-Risk / Developing / Optimizing / Leading)
- Use Salesforce Tasks + Chatter to notify reps within 15 minutes of a Tier 1 or Tier 2 completion from a named account

**Drift / Qualified (conversational assessment variant):**
- Build the assessment as a Drift bot playbook rather than a static form; this increases completion rates by 20-30% for high-intent website visitors
- Route Tier 1 completions instantly to a live rep or AI SDR for real-time conversation

**Notion + Zapier (lightweight / no-code stack):**
- Build the assessment in Tally (free Typeform alternative)
- Use Zapier to push results into a Notion database organized by tier
- Use Zapier to trigger personalized Gmail sequences via a templated draft for manual review

**AI Agent Automation (advanced):**
- Feed assessment responses into an AI agent (Claude, GPT-4o) via API to generate fully personalized report copy in real time — unique to each respondent, not templated
- Use an AI agent to draft the sales alert Slack message with personalized talking points and suggested opening line for the rep
- Quarterly: feed aggregate assessment data to an AI analysis prompt to identify emerging ICP pain trends and recalibrate scoring weights

## Troubleshooting

**Problem: Low completion rate (<50%) — too many people start but don't finish**
Solution: Audit question 3-5 (where most abandonment happens). Common culprits: (1) A question requires internal data the buyer doesn't have memorized — replace with qualitative perception options. (2) Questions feel like a product qualification survey, not a genuine diagnostic — rewrite using buyer-language pain framing. (3) The assessment is too long — cut to 8 questions maximum and collapse dimensions to 3.

**Problem: Leads are converting but sales says quality is poor — ICP mismatch**
Solution: Add one "silent qualifier" question that doesn't display in the report but captures company size or budget signal (e.g., "How many [relevant metric] does your team manage?" with answer ranges that proxy for company stage). Use this answer to add an ICP fit score field in HubSpot that gates whether an assessment lead flows to Sales vs. stays in marketing nurture.

**Problem: Assessment report doesn't feel personalized — buyers say it seems generic**
Solution: The report is likely using only the composite tier score rather than individual answer responses. Add conditional logic using answer-level branching: if they answered "A" to Question 2 AND "B" to Question 6, their report section on Dimension 1 displays a specific sentence variant. Most assessment tools (Typeform, Outgrow, ScoreApp) support answer-level conditional output. Alternatively, use an LLM API call at report generation time to write a unique summary paragraph incorporating their specific top 3 answers.
