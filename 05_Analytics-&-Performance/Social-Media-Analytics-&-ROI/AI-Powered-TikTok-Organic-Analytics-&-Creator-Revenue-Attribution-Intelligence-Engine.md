## Overview
This prompt transforms raw TikTok organic performance data into revenue-attributable intelligence — identifying which content formats, creators, hooks, and posting patterns drive measurable pipeline and revenue outcomes. Use it weekly to optimize your organic TikTok strategy and prove ROI to leadership without relying solely on paid media metrics.

## Integration Tips

- **Google Analytics 4**: Create a dedicated TikTok segment (source contains "tiktok") + configure UTM parameters on bio link (utm_source=tiktok&utm_medium=organic&utm_campaign=[content_type]). Use GA4's Path Exploration report to trace TikTok sessions to conversion.
- **TikTok Analytics API**: Export raw data via TikTok Business API or use tools like Sprout Social, Iconosquare, or Pentos for automated weekly data pulls. Feed structured CSV into this prompt monthly.
- **HubSpot**: Create a TikTok "Original Source" property using UTM tracking. Build a dashboard showing TikTok Organic as a lead source, tracking MQL/SQL rates from TikTok-sourced contacts.
- **Shopify**: Use the TikTok Pixel + Shopify app for Shop purchases; supplement with a dedicated promo code per creator/campaign. Pull order reports filtered by discount code for ground-truth revenue data.
- **Notion / Google Sheets**: Build a Content Intelligence Database logging each video's performance metrics, hook formula used, content category, and revenue fingerprint score. Update weekly and feed to this prompt for trend analysis.
- **Zapier / Make**: Automate weekly performance pulls from TikTok Analytics → Google Sheets → trigger this prompt via Claude API → output to Notion or Slack #marketing-analytics channel.
- **Stan.store / Beacons**: If selling digital products, integrate TikTok bio links through these platforms for cleaner attribution than generic link shorteners. Both offer native TikTok analytics integrations.

## Troubleshooting

**Issue: TikTok Analytics shows high views but GA4 shows minimal TikTok traffic.**
*Solution*: TikTok users frequently copy URLs and visit directly (dark social) or search the brand after watching (branded search). Cross-reference branded search volume in Google Search Console with video posting dates — a 20-40% branded search spike in the 48 hours post-viral video is a strong dark social signal. Add this to your attribution model.

**Issue: Engagement rates are high but zero revenue attribution.**
*Solution*: Your content is in the "Reach Builder" quadrant — entertaining but not driving conversion intent. Audit your top 10 posts: if fewer than 30% include a clear CTA (bio link reference, promo code, product mention) you have a call-to-action gap, not a content quality gap. Add soft CTAs to 50% of posts and measure link click rate changes over 30 days.

**Issue: Creator content performs inconsistently — impossible to predict what works.**
*Solution*: Run a "Content Variable Isolation" experiment: hold the hook format constant and vary only one element per week (format, topic, CTA style, video length). After 8 weeks you'll have a statistically meaningful performance signal per variable. Most brands make the mistake of changing multiple variables simultaneously, making it impossible to attribute performance deltas.
