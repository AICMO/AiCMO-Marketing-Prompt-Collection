# Influencer Marketing Analytics & Creator ROI Intelligence Engine - Full-Program Performance Measurement & Budget Optimization

**Difficulty:** Advanced | **Time:** 20-30 min | **Tags:** influencer-marketing, creator-economy, analytics, roi, attribution, b2b, b2c, automation

## Overview
Transforms raw influencer campaign data across creators, tiers, and platforms into actionable ROI intelligence — scoring individual creator performance, calculating true program-level return, reallocating budget toward proven performers, and generating board-ready influencer investment reports. Use it monthly or quarterly to optimize an ongoing creator program, not just analyze a single campaign.

## Integration Tips

- **Triple Whale / Northbeam / Rockerbox**: These attribution platforms natively track influencer promo codes and UTM paths across sessions. Feed the per-creator conversion data directly into the Module 2 attribution calculation. Use their "influencer" channel filter to pull the assisted conversion data for Layer 2 ROI.
- **Grin / Aspire / Modash**: Export creator performance reports as CSV. Map columns to the Creator Roster Data fields in the Advanced prompt. Most platforms auto-calculate EMV — use their number or recalculate with the CPM formula above and note the methodology difference.
- **HubSpot**: Tag all influencer-sourced contacts with utm_source=influencer in HubSpot contact properties. Build an Influencer Attribution Report in the Revenue Analytics tool filtered by this source. Map to Deals to capture pipeline influence for B2B programs.
- **Google Analytics 4**: Pull the "Session source/medium" report filtered to influencer UTMs. Use the "Conversion paths" exploration to identify multi-step paths that include influencer as a touchpoint for the Layer 2 assisted conversion calculation.
- **Shopify**: Install UTM tracking app (e.g., Lifetimely, TrueProfit) to associate UTM parameters with customer lifetime value — not just first-order revenue. This elevates influencer ROI when cohort LTV exceeds AOV.
- **Notion**: Build a Creator Roster Database with fields for CPS score, tier verdict, contract renewal date, and budget allocation. Automate a Zapier workflow to trigger a Slack alert 30 days before any creator contract expires with their CPS score attached.
- **Google Sheets**: Paste the Creator Roster Data table into a master tracking sheet. Add formulas for CPS auto-calculation using the scoring rubric. Build a quarterly trend chart showing program ROI across all three attribution layers by period.

## Troubleshooting

**Problem: "Most of our influencer conversions are untracked — we don't have UTM data for 60% of posts."**
Solution: Start immediately with mandatory UTM links in all new contracts (non-negotiable clause). For historical untracked posts, use the Dark Social Correction in Module 4: apply a 25-35% uplift to organic direct traffic during campaign periods as a conservative influencer credit estimate. Present this as "minimum floor" ROI, not total program value. For promo codes on untracked creators, use code redemption data as the primary attribution source going forward.

**Problem: "Our macro influencer drove huge reach but our CPA looks terrible compared to nano creators. How do I explain this to finance?"**
Solution: Apply the EMV calculation explicitly for macro tier — large-reach creators deliver brand awareness value that doesn't convert immediately but drives branded search, direct traffic, and future conversion lift. Present macro creators using EMV leverage ratio ($ media value / $ spend) rather than CPA. Benchmark: macro campaigns at 3-5x EMV leverage are considered efficient even with high direct-response CPA. Simultaneously, rebalance future budget toward micro/nano for performance goals and use macro selectively for launches and brand moments.

**Problem: "Our creator roster has grown to 40+ creators and this analysis is overwhelming — where do I start?"**
Solution: Apply the 80/20 filter first: rank all creators by UTM revenue + EMV combined, identify the top 20% (8 creators). Run the full Module 1-6 analysis only on those 8 first. For the bottom 80%, apply a simplified score using just engagement rate vs. benchmark and CPA vs. target — make a binary keep/exit decision. This reduces analysis to manageable scope while protecting your highest-value relationships.
