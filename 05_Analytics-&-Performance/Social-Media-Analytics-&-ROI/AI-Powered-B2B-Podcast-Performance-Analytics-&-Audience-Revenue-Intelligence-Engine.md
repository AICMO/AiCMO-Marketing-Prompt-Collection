## Integration Tips

**Analytics & Attribution Tools:**
- **Chartable / Spotify for Podcasters / Apple Podcast Connect**: Pull episode-level download data, completion rates, platform breakdowns, and listener geography into a structured data format for this prompt
- **HubSpot**: Create a custom "Podcast" source property in contact/deal records; use workflow automation to tag contacts who click podcast UTMs; build a "Podcast-Influenced Pipeline" report using attribution reporting
- **Salesforce**: Create a custom Campaign for each podcast episode; use Campaign Influence feature to track which opportunities have podcast episode campaigns in their journey
- **Zapier**: Connect Chartable webhooks → HubSpot/Salesforce to automatically create CRM activities when tracked listeners click show notes CTAs

**Workflow Automation:**
- Set up Google Looker Studio dashboard pulling from: Spotify API + HubSpot API + Google Sheets episode tracker → unified podcast performance view
- Automate weekly episode performance report: new episode publishes → 7-day download data auto-populated into reporting template → AI analysis triggered
- Use Clay or Apollo.io to enrich listener email captures (from show notes landing pages) with company data and ICP scoring before routing to CRM

**ABM Integration:**
- Cross-reference 6sense / Bombora intent data with podcast listener segments: companies showing high intent AND consuming podcast content = prioritize for high-touch outreach
- For Demandbase users: create a "Podcast Listener" audience segment and enable personalized ad retargeting for these high-intent accounts

## Troubleshooting

**Problem: No data on who is actually listening — only aggregate download numbers from hosting platform**
Solution: Implement a listener intelligence strategy: (1) Create an episode-specific resource download (checklist, template) requiring email capture — this de-anonymizes 3-8% of listeners; (2) Add "How did you hear about us?" to all conversion forms; (3) Run a quarterly listener survey via Typeform promoted across 3 consecutive episodes with a gift card incentive; (4) Use a podcast-specific UTM taxonomy in all show notes links to track website behavior

**Problem: Leadership doesn't believe podcast attribution data — "people don't buy because of a podcast"**
Solution: Shift from "podcast as direct source" to "podcast as influence multiplier" framing. Pull closed-won deal data and look for podcast touchpoints anywhere in the journey (not just as source). Run a cohort analysis: buyers who consumed 2+ podcast episodes before purchase vs. those who didn't — compare deal size, sales cycle length, and win rate. This data typically shows podcast-influenced deals close 18-35% faster with 12-22% higher ACV, which is a compelling business case that doesn't require podcast to be "the source."

**Problem: Episode performance is inconsistent — no clear pattern in what drives downloads**
Solution: You likely have a content strategy problem masked as an analytics problem. Implement the PACE scoring framework retroactively across your last 20 episodes. If no pattern emerges from topic/format/guest analysis, the issue is distribution, not content — audit your show notes SEO, cross-promotion strategy, and guest amplification agreements. Most B2B podcasts plateau not from content quality issues but from discovery failure: 80% of new listeners find shows through existing listeners' recommendations, not organic search.
