Maps the complete B2B buyer journey from first anonymous touchpoint to closed-won revenue, identifies conversion drop-off points across every funnel stage, and generates a prioritized optimization plan to accelerate pipeline velocity and improve win rates. Use this when MQL-to-SQL conversion is declining, pipeline velocity is stalling, or when you need to justify marketing spend with a journey-level attribution story for the CFO or board.

## Integration Tips

- **HubSpot:** Use the "Contact Activity" and "Deal Stage History" reports as your primary data source. Export lifecycle stage date fields to calculate time-in-stage. Use HubSpot's Attribution Reports to populate the source journey table. Set up Workflows to tag "golden path" contacts with a custom property for ongoing tracking.
- **Salesforce:** Run a SOQL query joining Leads, Contacts, Opportunities, and CampaignMembers tables to build the full journey dataset. Use Salesforce's Path Analytics (Einstein Analytics) or Tableau CRM to visualize stage transitions. Set up Process Builder alerts for the "intent signal" quick-win action.
- **Google Analytics 4:** Use GA4's Funnel Exploration report to map anonymous visitor journey through web touchpoints. Export the "Paths" report to identify the sequence of pages visited before form completion. Connect GA4 to BigQuery for deeper path analysis across sessions.
- **6sense / Bombora:** Layer intent data into your source quality analysis — accounts showing 3rd-party intent should be segmented separately to measure how intent-triggered journeys compare to non-intent journeys in conversion rate and velocity.
- **Gong / Chorus:** Feed call recording insights into the Opp-to-Close analysis. Ask Gong AI to identify the most common objection in lost deals at the late funnel stage — this directly informs your late-stage content gap recommendations.
- **Looker / Tableau:** Build a persistent journey dashboard with 4 views: (1) Funnel waterfall with benchmark lines, (2) Source quality matrix (volume vs. ACV scatter plot), (3) Content influence heatmap by journey stage, (4) Win/Loss trend by journey path. Refresh weekly.
- **Zapier / Make:** Automate the "intent signal alert" action — trigger a Salesforce task or Slack notification to an SDR when a known contact views the pricing page, ROI calculator, or case study 2+ times in 7 days.

## Troubleshooting

**Problem: Data is fragmented across HubSpot, Salesforce, and GA4 with no unified view — journey analysis is incomplete.**
Solution: Start with the data you have. Use CRM stage date fields as your primary source and manually build a stage transition table in Google Sheets. Don't let perfect be the enemy of good — a rough waterfall with estimated conversions still reveals the biggest leaks. Use this analysis to build the business case for a CDP or BI tool investment.

**Problem: Attribution model is disputed — marketing and sales disagree on which source gets credit, making source quality analysis politically fraught.**
Solution: Sidestep the attribution debate by presenting multiple views simultaneously: first-touch (shows what creates awareness), last-touch (shows what closes), and linear (shows what contributes). Frame the discussion around "journey quality by source" rather than "which channel gets credit." Showing that partner referrals have 34% higher ACV is inarguable regardless of attribution model.

**Problem: Sample sizes are too small — fewer than 20 closed-won deals in the analysis period, making conversion rate analysis unreliable.**
Solution: Extend the analysis window to 6-12 months to increase sample size. For win rate analysis specifically, use a rolling 12-month view even if other metrics use a shorter period. Flag in your report that conversion rates at small sample sizes have a ±5-10 point confidence interval. Focus the optimization roadmap on process changes (content gaps, alert triggers, sales follow-up speed) rather than channel cuts when data is thin.
