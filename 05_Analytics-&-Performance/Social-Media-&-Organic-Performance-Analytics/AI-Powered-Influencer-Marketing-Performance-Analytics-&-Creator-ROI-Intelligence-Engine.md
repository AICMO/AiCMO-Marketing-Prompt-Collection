Use this when leadership questions influencer ROI, when you need to cut underperforming creators from budget, or when scaling a creator program and must prioritize investment allocation.

## Troubleshooting

**Problem: Most conversions are unattributed — promo code redemption rate is <15%**
Fix: Promo codes alone fail when creators don't verbally call them out or when the audience is browsing on mobile and clicking links instead. Add a frictionless post-purchase survey (1 question: "How did you find us?") using Fairing or Enquire Labs. Simultaneously, implement UTM links in bio (Linktree or direct link) as your primary attribution path. Set a content brief requirement that every creator must mention the promo code twice — once verbally, once as a text overlay. If you're running on TikTok, add the promo code to the first comment pinned to each post.

**Problem: Micro-creators outperform on ROAS but leadership insists on macro/celebrity creators for "brand building"**
Fix: Build a dual-metric reporting framework that separates brand investment (macro/celebrity spend) from performance investment (micro spend) and measures each against different KPIs. For brand campaigns, measure Branded Search Volume lift (Google Search Console, 30-day window post-campaign), Direct Traffic lift, and consumer survey aided brand awareness. For performance campaigns, measure ROAS, CPA, and CAC. Present the portfolio as a deliberate strategy: macros build the audience, micros convert it. Use a holdout group (pause micro program in one region for 4 weeks) to prove the lift macro creators generate for downstream micro conversion.

**Problem: Creator data lives in 5 different spreadsheets, an influencer platform, and email threads — can't get a single view**
Fix: Consolidate into a master Airtable or Notion creator database immediately. Columns: Creator Handle, Platform, Tier, Monthly Fee, Content Deliverables, Unique Promo Code, UTM Tag, 90-Day Conversions, 90-Day Revenue, ROI%, Quadrant Classification, Contract Renewal Date. Pull data from your influencer management platform (AspireIQ/Grin exports), your eCommerce/CRM platform (discount code redemption reports), and your analytics platform (UTM-filtered conversion reports). Automate weekly data refreshes using Zapier or the platform's native API. Schedule a monthly creator performance review meeting with this database as the single source of truth.
