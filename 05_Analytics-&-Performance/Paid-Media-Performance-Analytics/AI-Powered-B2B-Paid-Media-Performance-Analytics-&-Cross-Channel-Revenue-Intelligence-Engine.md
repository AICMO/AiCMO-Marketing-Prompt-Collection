## Integration Tips

- **Google Ads → Salesforce/HubSpot:** Use native connectors (HubSpot Ads tool or Salesforce Marketing Cloud Connect) to import Google Ads cost data into CRM opportunity records. Enable offline conversion imports in Google Ads to send back CRM-verified SQL and closed-won signals — this trains Google's Smart Bidding on real revenue, not just form fills.
- **LinkedIn Campaign Manager → CRM:** Use LinkedIn's Insight Tag + Conversions API (CAPI) for cookieless tracking. Map LinkedIn Lead Gen Forms directly to HubSpot/Salesforce via Zapier or native integration. Import Matched Audiences from CRM monthly (closed-won list, churned list, current customers) to suppression lists.
- **Unified Reporting in Looker Studio / Tableau:** Pull all channel spend via API (Google Ads API, LinkedIn Marketing Developer Platform, Meta Marketing API) into a single BigQuery or Snowflake table. Join to CRM opportunity data on UTM parameters + form submission timestamps to build cross-channel pipeline attribution table.
- **Slack Anomaly Alerts:** Configure AI agent to post weekly performance digest + anomaly alerts to a #paid-media-alerts Slack channel. Use Zapier or n8n to trigger alerts when any metric crosses defined thresholds (e.g., CPL spikes >20%, a campaign burns >$500/day with 0 form fills).
- **Budget Automation via Rules:** Use Google Ads Automated Rules and LinkedIn Campaign Manager budget rules for pacing. For cross-channel pacing, use Optmyzr, Supermetrics + Google Sheets, or a custom Python script on a daily cron job to redistribute budget based on CPL-to-target ratios.
- **Quarterly Business Review:** Export the Program Health Scorecard section as a slide deck input using Beautiful.ai, Gamma, or a custom Notion template. Link CRM opportunity data from the period so finance can independently verify revenue attribution.

## Troubleshooting

**Problem:** Platform-reported conversions are 3–5x higher than CRM-verified pipeline, making it impossible to trust channel performance rankings.
**Solution:** This is universal in B2B paid media and indicates attribution overlap (the same form fill being claimed by Google, LinkedIn, and email simultaneously). Fix by: (1) implementing UTM governance across all channels using a consistent taxonomy; (2) enabling Salesforce/HubSpot as the system of record for all attribution, not ad platforms; (3) switching Google Ads and LinkedIn to import offline conversions from CRM rather than using pixel-based view-through attribution. Accept that platform dashboards will show lower numbers — they're now accurate.

**Problem:** LinkedIn CPL is 4–6x Google CPL, causing leadership to question the channel's value despite LinkedIn generating more enterprise pipeline.
**Solution:** CPL is the wrong metric for LinkedIn enterprise targeting. LinkedIn reaches high-ACV prospects (VPs, CROs) who are not actively searching — its value is early-stage pipeline creation, not cost-efficient form fills. Reframe the analysis: calculate Pipeline-Per-Dollar and Closed-Won-CAC by channel. LinkedIn's higher CPL is often offset by higher ACV deals and faster sales cycles. Present a side-by-side comparison of Average Deal Size and Close Rate by channel origin to make the case.

**Problem:** The AI-generated optimization recommendations conflict with what the media buyer is seeing in the platform interfaces.
**Solution:** The most common cause is lag in CRM data — paid media platforms update in real time while CRM data reflects a 15–45 day lag to opportunity creation. Always specify a data-lag adjusted period (use leads from 45+ days ago to evaluate MQL and SQL conversion rates; use current-week data only for spend pacing and CTR anomalies). Build this lag correction into the AI agent's analysis instructions as a standing rule.
