## Integration Tips

- **Google Ads:** Use the Google Ads API (or Google Ads Scripts) to pull campaign-level spend, clicks, conversions, and Search Impression Share daily into a BigQuery or Google Sheets staging table. Schedule this as an automated agent trigger at 7 AM.
- **LinkedIn Campaign Manager:** Connect via LinkedIn Marketing Developer Platform API. Export campaign analytics at the ad and audience level. Enable Insight Tag for website retargeting and attribution.
- **Meta Ads Manager:** Use Meta Marketing API with `action_attribution_windows: ['7d_click']` explicitly set — never use default 7d_click + 1d_view for B2B SaaS (inflates 3–10x). Pull via CAPI (Conversions API) rather than pixel for cookieless accuracy.
- **HubSpot:** Create a HubSpot Workflow that stamps "Paid Source" on every contact whose original/latest source contains UTM parameters from your paid channels. Build a Custom Report for Cost per MQL by UTM Source using the HubSpot Ads ROI Report.
- **Salesforce:** Use Salesforce Campaign Influence (Campaign Statistics) to track first-touch and multi-touch revenue attribution. Sync UTM parameters to the Lead Source and Lead Source Detail fields. Build a Salesforce Report: Opportunities closed this period, grouped by Lead Source, showing total ARR.
- **Unified Dashboard (Looker / Tableau / Supermetrics):** Connect all platform APIs via Supermetrics → Google Sheets, then pipe to Looker Studio or Tableau. Build a single "Paid Media P&L" view: Spend → MQLs → SQLs → Pipeline → Revenue per channel.
- **Slack Alerts:** Use Zapier or Make to send daily performance digests to your #paid-media-alerts channel: any campaign with 0 conversions and >$200 spend, any metric >20% off target.
- **Budget Pacing:** Connect to Optmyzr or Skai for automated daily budget pacing alerts. Set rules: if Monday spend pace suggests you'll exhaust budget by Thursday, auto-alert and suggest bid adjustments.

## Troubleshooting

**Problem: Platform-reported conversions are 5–10x higher than CRM-verified MQLs**
Solution: This is attribution inflation. Check three settings immediately: (1) Meta — switch from 7d_click+1d_view to click-only window in Ad Settings → Attribution; (2) LinkedIn — audit whether your Lead Gen Form qualifying questions filter for ICP or accept any email; (3) Google — check if "all conversions" includes micro-conversions (page views, scroll depth) mixed with form completions. Separate conversion actions in Google Ads and only optimize campaigns toward high-intent conversions (demo requests, form completions). Re-run this prompt with click-only attribution data.

**Problem: Budget reallocation recommendations conflict with brand-awareness goals from leadership**
Solution: Split the budget analysis into two tracks: (1) Performance budget — measure purely on cost per MQL and pipeline ROI, optimize ruthlessly; (2) Brand budget — measure on reach, frequency against ICP, and brand lift (LinkedIn Brand Lift study or Google Brand Lift Survey). In the Advanced Version, specify "Performance Budget: $[X] | Brand Budget: $[X]" and analyze each separately. Present CFO with performance ROI; present CMO with brand health metrics separately.

**Problem: No CRM attribution data — only platform-reported conversions available**
Solution: This is a critical gap. As an immediate workaround, apply industry-standard "B2B platform inflation factors" as correction multipliers: LinkedIn Lead Gen Forms → 8–12x inflation factor (divide form completions by 10 to estimate true MQLs); Meta Conversions → 15–40x inflation factor for B2B (divide Meta conversions by 20 as baseline estimate); Google Search → 1.5–2x inflation (most reliable but still overcounts). Run the prompt with these adjusted estimates, label all outputs as "estimated" with [Low confidence], and prioritize implementing UTM-to-CRM tracking (HubSpot UTM tracking or Salesforce Campaign Source mapping) within the next 30 days as Priority 1 MarTech fix.
