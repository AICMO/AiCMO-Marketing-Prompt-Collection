## Overview
Analyzes organic search performance across keyword clusters, content assets, and funnel stages to identify SEO revenue contribution, detect ranking decay, surface pipeline attribution from organic, and generate a prioritized optimization roadmap. Use this when you need to justify SEO investment to the CFO, diagnose why organic traffic is converting at a lower rate, or build the case for additional content resources.

## Success Metrics

- **Organic pipeline ROAS** — primary optimization metric; B2B SaaS benchmark: 15-40x for optimized programs
- **Organic CPL vs. blended paid CPL** — organic should be 10-25x cheaper; if not, your content is not ranking for commercial intent
- **Ranking velocity** — % of tracked BOFU keywords moving to positions 1-5 over 90-day period
- **Content payback period** — time from content publication to first pipeline attribution; target < 90 days for BOFU pages, 6-12 months for TOFU
- **Organic pipeline contribution %** — mature B2B SaaS programs generate 25-40% of pipeline from organic; below 15% signals structural underinvestment
- **Forecast accuracy** — AI-projected vs. actual organic pipeline within 20% at 60-day checkpoint

## Integration Tips

- **Google Search Console:** Export the "Search results" report filtered to your top 100 pages (last 90 days). Download the "Pages" tab CSV for session-level data. Cross-reference with your CRM lead source data to build the pipeline attribution column manually before analysis — this one-time setup enables the most powerful outputs.
- **HubSpot:** Use the "Traffic Analytics" report filtered to "Organic Search" source. Enable HubSpot's "Content Strategy" tool to see keyword ranking data alongside contact attribution. Build a custom report: Contacts (by Original Source = Organic Search) → filtered to MQLs → pipeline value.
- **Salesforce + Google Analytics 4:** Use GA4's "Landing page" report filtered to organic session source, then join with Salesforce opportunity data via a shared "original source" field. Google Looker Studio can automate this join for a live dashboard.
- **Ahrefs / Semrush:** Export the "Organic Pages" report with estimated traffic value, then overlay with your CRM pipeline data by URL. Use the "Keyword Gap" feature to generate the competitor gap analysis section input directly.
- **Zapier + Google Sheets:** Build a weekly automation: GSC data → Google Sheets → Claude API with this prompt → executive summary posted to Slack #seo-weekly channel every Monday. With Sheets Apps Script, you can auto-refresh the GSC data pull on a schedule.
- **Notion:** Store the rolling output of this prompt in a Notion database with tags for "priority," "status," and "owner." Use Notion's AI to summarize trends across monthly reports and flag when recommendations from prior months weren't implemented.

## Troubleshooting

- **Problem:** Pipeline data from organic isn't available because CRM lead source attribution is incomplete or inconsistent.
  **Solution:** Start with a proxy metric: use Google Analytics 4 "Goals" (demo requests, pricing page visits, trial signups) attributed to organic sessions. Multiply goal completions by your blended MQL-to-pipeline conversion rate and ACV to estimate pipeline. Then fix source tracking at the form level — add UTM parameter capture to all HubSpot/Salesforce forms so organic sessions that convert carry proper source data. This is a 2-week fix with outsized analytical impact.

- **Problem:** The AI recommends investing in TOFU content, but leadership only wants BOFU pipeline.
  **Solution:** Reframe the analysis using the "organic content funnel math": TOFU content at scale creates the brand familiarity that makes BOFU content convert at 2-3x the rate. Run the attribution analysis section and show multi-touch data — in most B2B orgs, 40-60% of pipeline that closes via BOFU organic content was first touched by a TOFU organic post 3-9 months earlier. If multi-touch data isn't available, present the competitor comparison: check if your top competitors publish high volumes of TOFU content (they almost certainly do) and frame TOFU as competitive table stakes, not a nice-to-have.

- **Problem:** Rankings fluctuate week-to-week making it hard to identify true decay vs. normal volatility.
  **Solution:** Use 90-day rolling averages, not point-in-time snapshots. In the prompt, specify: "calculate average position as a 90-day rolling average, not a single-day reading." Flag a page as decaying only if it shows a consistent downward trend across three consecutive 30-day periods AND the decline correlates with a drop in sessions (not just ranking volatility). Google's algorithm updates cause 1-3 week volatility that self-corrects; true decay is structural and shows up in both rankings and traffic simultaneously.
