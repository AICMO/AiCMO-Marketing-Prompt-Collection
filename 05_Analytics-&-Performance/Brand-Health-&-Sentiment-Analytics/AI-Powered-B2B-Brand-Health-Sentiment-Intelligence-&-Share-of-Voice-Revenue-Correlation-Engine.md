## Integration Tips

**G2 & Review Platforms:**
- Connect G2 Buyer Intent API to HubSpot to trigger "brand awareness nurture" sequence when target accounts visit your G2 profile without converting
- Use G2's review export API to feed sentiment data into your brand health dashboard in Google Sheets/Looker Studio
- Set up Zapier webhook: new G2 review → Slack alert to CS team → auto-tag in Salesforce account record

**Salesforce:**
- Add custom field "Brand Objection Type" to Opportunity object (picklist: Awareness / Trust / Sentiment / SOV / None)
- Create Salesforce report: "Losses by Brand Objection Type" — refresh monthly and feed into this prompt
- Build dashboard tile: "Win Rate: Brand-Aware Leads vs. Cold Outbound" using Lead Source + Opportunity fields

**HubSpot:**
- Create Smart List: contacts who visited /customers, /case-studies, or /reviews pages more than twice (brand validation behavior) — track conversion rate to SQL separately
- Workflow: If NPS response < 7, delay G2 review request 60 days; if NPS ≥ 8, trigger review ask within 48 hours
- Use HubSpot Social tool to track branded mention volume week-over-week

**Brandwatch / Mention / Sprout Social:**
- Set up Boolean query for: [Brand Name] AND (review OR "worth it" OR "alternative" OR "vs [Competitor]") — feed results monthly into this prompt's social sentiment section
- Export SOV data as CSV and paste directly into the Advanced Version's Share of Voice input block

**Notion / Google Sheets Dashboard:**
- Build a "Brand Health Weekly" tracker: columns for each scorecard dimension, populated from tool exports
- Use Google Sheets IMPORTXML to auto-pull your G2 rating weekly without manual updates

## Troubleshooting

**Problem: "I don't have enough data to run the advanced version — we don't track most of these signals."**
Solution: Start with the Quick Copy-Paste Version using only what you have (even 2-3 signals). In the output, the agent will generate a prioritized data collection plan as a deliverable. Run that plan for 30 days, then rerun with more data. Minimum viable input: G2 score + recent review themes + 2-3 sales loss themes from your CRM.

**Problem: "The revenue correlation numbers feel made up — how do I make them credible?"**
Solution: Anchor all estimates to your own CRM data before running the prompt. Export: (1) win rate by lead source, (2) average sales cycle by deal source, (3) discount frequency by deal stage. Paste these into the Company Context block. The agent will use your actual numbers instead of industry benchmarks, making the correlation model defensible to a CFO.

**Problem: "The competitors I listed don't match what my buyers actually compare us against — the analysis feels off."**
Solution: Pull 30 days of loss reasons from your CRM and look for the competitor names that appear most often. Also check your G2 profile's "Compared To" section — G2 shows which competitors buyers compare you against on your profile page. Use those names instead of assumed competitors. This single change dramatically improves the competitive positioning analysis accuracy.
