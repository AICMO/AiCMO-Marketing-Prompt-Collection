## Overview
Analyzes the full content-led demand generation engine — attributing inbound pipeline to specific content programs (SEO/blog, webinars, gated assets, social, email newsletter) despite dark funnel complexity. Use this when you need to justify content marketing investment to a CFO, when last-touch attribution is undercounting inbound ROI, or when you need to prove that ungated thought leadership is driving pipeline even when it doesn't appear in the CRM.

## Integration Tips

- **HubSpot / Marketo**: The multi-touch attribution model in this prompt maps directly to HubSpot's position-based attribution model (available in Marketing Hub Professional+) and Marketo's Revenue Cycle Analytics. Enable multi-touch attribution before running this prompt to get accurate "influenced pipeline" numbers. If still on last-touch, use the self-reported survey data as a first-order correction factor.
- **6sense / Demandbase / Clearbit Reveal**: The dark funnel analysis requires site visitor identification. Use the 6sense Audience API or Clearbit Reveal webhook to log anonymous company-level content consumption events in a separate spreadsheet or data warehouse. Feed this data into the Signal B dark funnel calculation. Even 60 days of data is sufficient to estimate annual dark funnel pipeline.
- **Google Analytics 4**: Configure GA4 custom events to track content engagement depth: scroll depth (50%, 90%), time-on-page (2+ minutes), internal link clicks to pricing/demo pages, and return visits within 30 days. These behavioral signals map to the Content-to-Pipeline Conversion Waterfall stages and can be pushed to HubSpot as custom contact properties via the GA4-HubSpot integration.
- **Typeform / HubSpot Forms**: Add a 3-question self-reported attribution survey to every demo request, trial signup, and gated asset download form. Question 1: "How did you first hear about [Company]?" Question 2: "Which piece of content was most helpful in your evaluation?" Question 3: "What was the main problem you were trying to solve when you found us?" This single change typically closes 40-60% of the attribution gap within 90 days and costs nothing to implement.
- **Google Sheets / Airtable**: Build the Content Asset Pipeline ROI tracker as a simple spreadsheet: Column A = content asset name, Column B = publication date, Column C = production cost, Column D = monthly pipeline attributed (last-touch), Column E = monthly pipeline influenced (any-touch), Column F = pipeline ROI (auto-calculated). Update monthly and use to prioritize which assets to update, promote, or retire. Share with content team as the primary KPI dashboard for content production decisions.
- **Salesforce Revenue Intelligence / Clari**: Add "Content Influence Score" as a custom opportunity field in Salesforce — populated automatically from marketing automation engagement data. This enables the sales team to see which accounts have been deeply educated by content before the first sales call, allowing reps to skip basic education conversations and focus on differentiated value discussion.

## Troubleshooting

**Problem: I have almost no attribution data — no self-reported survey, no visitor identification tool, just HubSpot last-touch.**
Fix: Start with what you have and build upward. Run the Last-Touch Attribution calculation (Method A) as your baseline.
